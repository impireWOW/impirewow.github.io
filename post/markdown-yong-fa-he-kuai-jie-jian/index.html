markdown用法和快捷键

· 2022-02-17 · # 工具

Markdown 的目标是实现「易读易写」
下边简单列举一些markdown的使用方法

一,无序列表

使用方法
* 无序列表1
* 无序列表2
    * 无序列表2.1
    * 无序列表2.2

实现样式

  • 无序列表1

  • 无序列表2

    • 无序列表2.1
    • 无序列表2.2

**快捷键:**CTRL+U (UNORDERED)

二.有序列表:

使用方法
1.(空格)有序列表1
2.(空格)有序列表2
回车的时候会自动补码

实现样式

  1. 有序列表1
  2. 有序列表2

**快捷键:**CTRL+O ORDERED

三.标题:

使用方法:
# 一级标题
## 二级标题
.....
###### 六级标题
#后记得跟空格!

实现样式:

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

四.分割线

  1. 使用方法
  2. ***或者___

实现样式:

或者(其实都一样)

五.斜体字,粗体字

1. 使用方法
2. *斜体字*  ctrl+I
3. **粗体字**  ctrl+B
挨着**的地方不要有空格否则不识别

实现样式
斜体字
粗体字

六.添加注释

使用方法
>注释一段话
>添加引用的内容

实现样式

注释一段话
添加引用的内容

七.网页链接

使用方法
[网页名字](地址)

实现效果
网页名字
快捷键
CTRL+L

八.代码

使用方法:代码前后的行加入 ''' 即可
(数字键1旁边那个`键)

实现样式

假装有代码,不写代码也可以,分段用

快捷键 CTRL+K

九.表格

使用方法
标题|标题|标题
:--:|:--|--:
居中对齐|左对齐|右对齐
内容1|内容2|内容3

实现样式

标题 标题 标题
居中对齐 左对齐 右对齐
内容1 内容2 内容3

十.图片

方式1,直接拖进去
方式2,需要将图片上传到图床
[图片名称](图片地址url)

快捷键CTRL+G

十一,短文字

使用方法
`短不短`
(数字键1旁边那个`键)

实现样式
短不短

十二,删除文字

使用方法
~~双波浪删除内容~~

实现样式
双波浪删除内容

快捷键 : CTRL+R
