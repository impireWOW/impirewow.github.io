equals和==的区别

2022-02-14
中文 java 填坑

文章目录

==的含义: 基本数据类型:byte,short,char,int,long,float,double,boolean 如果使用双等号==来对比,比较的是他们的值

引用数据类型: 使用==对比的时候,比较的是他们在内存中的存放地址(堆内存地址) 一般除非是同一个new出来的对象,他们比较后的结果为true,否则比较后结果为false,因为每次new对象,都会重新开辟新的堆内存空间

public boolean equals(Object obj) {
//this - s1
//obj - s2
return (this == obj);
}

这个方法的初始默认行为是比较对象的内存地址 所以在一些类库中这个方法被重写了,如string,integer,date,这些类中equals有其自身的实现(一般是比较对象的成员变量值是否相同),而不是比较类在堆内存中的存放地址了
