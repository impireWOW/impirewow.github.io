equals和==的区别

· 2022-02-16 · # Java # JVM

==的含义:

基本数据类型:byte,short,char,int,long,float,double,boolean
如果使用双等号==来对比,比较的是他们的值

引用数据类型:
使用==对比的时候,比较的是他们在内存中的存放地址(堆内存地址)
一般除非是同一个new出来的对象,他们比较后的结果为true,否则比较后结果为false,因为每次new对象,都会重新开辟新的堆内存空间

equals()方法:

equals()源码:
public boolean equals(Object obj) {
//this - s1
//obj - s2
return (this == obj);
}

这个方法的初始默认行为是比较对象的内存地址
所以在一些类库中这个方法被重写了,如string,integer,date,这些类中equals有其自身的实现(一般是比较对象的成员变量值是否相同),而不是比较类在堆内存中的存放地址了

String类的equals()方法:

public boolean equals(Object anObject) {
    if (this == anObject) {
        return true;
     }
    if (anObject instanceof String) {
        String anotherString = (String)anObject;
        int n = value.length;
        if (n == anotherString.value.length) {
            char v1[] = value;
            char v2[] = anotherString.value;
            int i = 0;
            while (n-- != 0) {
                if (v1[i] != v2[i])
                    return false;
                 i++;
            }
            return true;
        }
    }
    return false;
}
