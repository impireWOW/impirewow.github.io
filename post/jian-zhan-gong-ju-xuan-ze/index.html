建站工具选择

2022-02-12
中文 工具

文章目录

很多新手会推荐使用宝塔面板 https://www.bt.cn/ 来进行网页服务器环境的初始化 宝塔自身的一键部署和图形化界面一开始看来确实是蛮好用的,但是也会额外的耗费一部分性能 而且一键化部署也对于源文件目录的操作不是很好 如果出了问题还是得去shell里修改配置

所以不如自己去使用LNMP https://lnmp.org/ 或者LAMP https://www.lamp.sh/ 来进行环境的部署 LNMP一键安装包是一个用Linux Shell编写的可以为CentOS/RHEL/Fedora/Aliyun/Amazon、Debian/Ubuntu/Raspbian/Deepin/Mint Linux VPS或独立主机安装LNMP(Nginx/MySQL/PHP)、LNMPA(Nginx/MySQL/PHP/Apache)、LAMP(Apache/MySQL/PHP)生产环境的Shell程序。

LAMP 一键安装包是一个用 Linux Shell 编写的可以为 Amazon Linux/CentOS/Debian/Ubuntu 系统的 VPS 或服务器安装 LAMP(Linux + Apache + MySQL/MariaDB + PHP) 生产环境的 Shell 脚本。包含一些可选安装组件如： Zend OPcache, ionCube Loader, PDFlib, XCache, APCu, imagick, gmagick, libsodium, memcached, redis, mongodb, swoole, yaf, yar, msgpack, psr, phalcon, grpc, xdebug

其他诸如：OpenSSL, ImageMagick, GraphicsMagick, Memcached, phpMyAdmin, Adminer, Redis, re2c, KodExplorer

同时还有一些辅助脚本如：虚拟主机管理、Apache、MySQL/MariaDB、PHP 及 PhpMyAdmin、Adminer 的升级等。

通过SSH和一键安装命令即可按照个人选择来进行环境配置 wget http://soft.vpser.net/lnmp/lnmp1.8.tar.gz -cO lnmp1.8.tar.gz && tar zxf lnmp1.8.tar.gz && cd lnmp1.8 && ./install.sh lnmp 如果想要安装LAMP 或者 LNPMA 把命令最后的 lnmp改成自己想要的就可以了

如果想要卸载就在下载的目录里运行uninstall.sh
